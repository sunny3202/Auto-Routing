# AutoRouting Client — Electron + Next.js

---

## 1. 기술 스택

| 항목 | 내용 |
| ---- | ---- |
| 프레임워크 | Next.js (SSG 모드, `output: "export"`) |
| 런타임 | Electron |
| 언어 | TypeScript |
| 상태관리 | React Context API |
| API 통신 | Fetch API + JWT Bearer |
| API 프록시 | `localhost:8010` (next.config.js rewrites) |
| IPC 브리지 | `window.ipc` (preload.d.ts 타입 정의) |
| 빌드 결과 | `distDir: "../app"` → `app/home.html` |

---

## 2. 페이지 워크플로우 (10개 페이지)

```
/home
  → 라우팅 타입 선택 (Vacuum / Exhaust)
  → TargetContext에 설정
  ↓
/input-page
  → fromTo 파일 선택 (JSON, 복수)
  → obstacle 파일 선택 (BIM JSON)
  → "Upload" 버튼 → FileUploadService → POST /upload/input/{historyId}
  ↓
/input-validation-progress-page
  → GET /progress/validation/{requestId} 폴링
  ↓
/input-validation-report
  → ValidationResultsTable (ERROR/WARNING/SUCCESS)
  → ERROR 없으면 다음 단계 진행 가능
  ↓
/rule-collection-page
  → RuleSetsContext 룰셋 목록
  → 편집 → /rule-editor/[id]
  ↓
/preprocessing-progress-page
  → GET /progress/preprocessing/{requestId} 폴링
  ↓
/preprocessing-report
  → 전처리 결과 표시
  ↓
/routing-progress-page
  → GET /progress/routing/{requestId} 폴링
  ↓
/routing-report
  → result_report.json 렌더링
  → result.json 다운로드 버튼
```

---

## 3. 상태 관리 구조 (Context)

| Context | 역할 |
| ------- | ---- |
| `TargetContext` | 선택된 라우팅 타입 (Vacuum/Exhaust) + 단계 |
| `InputContext` | 업로드된 파일 참조, requestId |
| `RuleSetsContext` | 룰셋 목록 + 파라미터 (routing_option.json 기반) |
| `GlobalProviders` | 위 Context 통합 제공 |

## 5. Electron 메인 프로세스 추가 (신규)

현재: `main/` 폴더 없음, `preload.d.ts` 타입 참조만 존재.

```javascript
// main/main.js (신규 작성)
const { app, BrowserWindow, ipcMain } = require('electron')
const { spawn } = require('child_process')
const path = require('path')

let serverProcess

app.on('ready', () => {
  // arithmetic-server 자동 실행
  serverProcess = spawn('python', ['-m', 'arithmetic_server'])

  const win = new BrowserWindow({
    webPreferences: {
      preload: path.join(__dirname, 'preload.js')
    }
  })
  win.loadFile(path.join(__dirname, '../app/home.html'))
})

app.on('quit', () => {
  if (serverProcess) serverProcess.kill()
})
```

```javascript
// main/preload.js (신규, preload.d.ts 타입 맞춤)
const { contextBridge, ipcRenderer } = require('electron')
contextBridge.exposeInMainWorld('ipc', {
  send: (channel, data) => ipcRenderer.send(channel, data),
  on: (channel, func) => ipcRenderer.on(channel, (event, ...args) => func(...args))
})
```
